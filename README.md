## Description
Practicando con Nest.js creando una Restful api con bases de datos.

## Installation

1. Clonar el repositorio.
2. Ejecutar comando:

```bash
$ yarn install
```

3. Instalar Nest CLI:

```bash
$ npm i -g @nestjs/cli
```

4. Desplegar DB Mongo con Docker

```bash
$ docker-compose up -d
```

### Requerimientos - Stack:
* mongodb
* Nestjs
